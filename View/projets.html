mpy-embeded

Projet personnel de découverte de MicroPython avec la Raspberry Pi Pico W, utilisant le Freenove Basic Starter Kit. Ce carnet de bord documente l’apprentissage progressif de la programmation embarquée à travers l’exploration des composants (LED, boutons, capteurs) et la réalisation de montages et scripts MicroPython simples, comme l’allumage de LED grâce à des boutons.

L’objectif principal est d’apprendre à programmer des microcontrôleurs avec MicroPython via des projets concrets et évolutifs. Le matériel utilisé inclut la carte Raspberry Pi Pico W (dual-core ARM Cortex-M0+), le kit Freenove, ainsi qu’une breadboard pour le prototypage rapide.

Weather

Programme Python qui récupère et affiche les informations météo d'une ville via l'API OpenWeatherMap. Il affiche la description du temps, la température en degrés Celsius, l’humidité et la pression atmosphérique, en traitant les données JSON récupérées par une requête HTTP.

Ce projet a été réalisé dans le cadre d’un exercice d’initiation à Python et aux API. Il m’a permis d’apprendre à gérer les entrées utilisateur, faire des requêtes HTTP avec requests, manipuler des données JSON, et présenter les résultats de façon claire, tout en me familiarisant avec le traitement des données et l’utilisation d’API externes.

Projets C/C++

embedded-digicode-projet

Projet de digicode avec un Raspberry Pi Pico utilisant un clavier matriciel pour saisir un code. Le système vérifie le code saisi par rapport à un code prédéfini et affiche un message de confirmation ou d’erreur via la console. La saisie se réinitialise automatiquement après 15 secondes d’inactivité.

Trois LEDs indiquent l’état du système : LED verte pour code correct, LED rouge pour code incorrect, et LED orange si le délai de saisie est dépassé. Le développement a été réalisé avec PlatformIO sous VSCode, facilitant la gestion et le déploiement du code sur le microcontrôleur. Le matériel utilisé comprend un Raspberry Pi Pico, un clavier matriciel, des LEDs et les câbles nécessaires au câblage.
